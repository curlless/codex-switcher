# US004: Build the Cursor-inspired profile workspace MVP

**Status:** Backlog
**Epic:** Epic 2
**Labels:** user-story
**Created:** 2026-03-07

## Goal

Implement the first fully usable desktop GUI workflow for browsing profiles, understanding usage state, and switching or reloading accounts from a polished workspace UI.

## Acceptance Criteria

1. Users can browse profiles, inspect profile details, and see active/reserved states.
2. Users can preview and execute switching from the GUI.
3. Users can trigger reload actions with clear outcome feedback.
4. The visual result follows the documented minimal dark desktop direction.

## Technical Notes

- This story depends on both the desktop shell and the extracted service layer.
- MVP focus is clarity and flow speed, not advanced settings.

## Validation Notes

- Keyboard navigation and empty/error states are part of MVP quality, not stretch goals.
